Simple and easy to use wordpress pluging to add the meta description tag in html head

== Description ==
This plugin lets the user write a seperate description for each post, page or custom post type post which is added as a meta description to the html head. 

It also adds meta description to category, tag and custom taxonomy archive as well as front page. For category, tag and custom taxonomy the Description you can edit on the Edit Category or Edit Tag pages in the Dashboard is used. For front page the Tagline under Settings > General Settings is used.

== Installation ==
Put the unzipped files in the plugin directory and go to Dashboard > Plugins to activate.
